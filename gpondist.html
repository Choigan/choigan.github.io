Основная проверка ONT.

Последовательность проверки. Сначала выводим список всех ONU и находим по мак-адрессу нужный нам интерфейс. Затем заходим в меню конфигурирования, открываем нужный интерфейс и проверяем сигнал.

show onu-status - вывести все ONU.

enable - включение режима конфигурирования.

configure terminal - вход в меню конфигурирования.

onu 0/N/ID - выбор интерфейса конфигурирования.

show onu-opm-diagnosis - параметры оптики.

Дополнительная проверка ONT.

show mac-address-table onu - вывести мак-адресса всех ONU.

show mac-address-table onu XX:XX:XX:XX:XX:XX08:C6:B3:2E:7B:69 - вывести мак-адресса всех ONU.

Основная проверка ONT.

Последовательность проверки. Сначала заходим в меню конфигурирования 0/0, выводим ONT во всех портах и находим нужный по мак-адрессу и проверяем сигнал оптики. Затем выходим в меню конфигурирования и проверяем мак-адресса устройства подключенного после ONT.

enable - включение режима конфигурирования.

config - вход в меню конфигурирования.

interface epon 0/0 - выбор интерфейса конфигурирования.

show ont optical-info N all - параметры оптики.

exit - выход до предыдущего меню.

show mac-address all - вывести все мак-адресса.

Дополнительная проверка ONT.

show ont optical-info N ID - полная информация оптического сигнала.

show mac-address ont 0/0/N ID - вывести мак-адресса выбранного интерфейса.

Ниже команды требуют проверки, примеры взял из интернета

show mac-address ont 0/0/N - вывести мак-адресса выбранного порта.

show mac-address vlan {vlan}0 - 4095 - вывести мак-адресса с выбранного vlan.

show mac-address dynamic include XX:XX:XX:XX:XX:XXCC:9D:A2:D5:CB:C4 - найти по мак-адрессу.

show ont distance N all - полная информация оптического сигнала.

ont reboot N ID - перезагрузка терминала.

ont deactivate N ID - деактивация терминала.

ont activate N ID - активация терминала.

mac-address flush all - очистка таблицы мак-адрессов.

show vlan all - просмотр всех VLAN.
