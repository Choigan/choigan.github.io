Основная проверка IES-1248.

statistics adsl linerate N - просмотр скорости линии.

statistics mac N - просмотр мак-адресса.

statistics adsl show N - просмотр UP-TIME.

Основная проверка IES-3000.

lc com N linerate ID - просмотр скорости линии.

lc red N wddi mac show - просмотр мак-адрессов.

lc com N show port ID - просмотр UP-TIME.

Дополнительная проверка IES-1248.

statistics mac - отображение всех мак-адрессов.

statistics adsl lineinfo N - просмотр параметров линии.

adsl show N - просмотр профиля порта.

Команды IES-1248.

adsl disable N - отключение порта.

adsl enable N - включение порта.

switch vlan portshow N - просмотр настроек VLAN на порту.

switch vlan pvid N {vlan}0 - 4095 - присваивание VLAN на порту.

switch vlan show * - просмотр настроек VLAN всех портов.

adsl profile map N {профиль}Название профиля auto - назначение профиля к порту.

adsl profile show - просмотр всех профилей.

adsl profile show {профиль}Название профиля - просмотр параметров выбранного профиля.

adsl profile set {профиль}Название нового профиля interleave 1024Исходящая скорость 2048Входящая скорость 6 0 31 32 6 0 31 64 3 9 3 9 - создание нового профиля.

adsl profile delete {профиль}Название удаляемого профиля - удаление профиля.

mac flush - очистка таблицы мак-адрессов.

switch mac count show N - просмотр количества разрешенных мак-адрессов на порту.

switch mac count set N {0-5} - установка количества разрешенных мак-адрессов на порту.

adsl pvc show N - просмотр виртуальных каналов порта.

statistics port N - просмотр статистики порта.

sys date show - просмотр даты системы.

sys time show - просмотр времени системы.

sys info show - просмотр информации.

sys log show - просмотр журнала событий системы.
