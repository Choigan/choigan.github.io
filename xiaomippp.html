Настройка PPPoE на XIAOMI

Открываем любой браузер на устройстве, которое подключено по Wi-Fi или с помощью кабеля к роутеру.

В адресной строке пишем 192.168.31.1 и нажимаем Enter.

Откроется главное меню роутера. Выбираем страну Russia (1), ставим галочки напротив: Select your country or region of use и Read and agreed to Mi Wi Fi Router User Agreement (2). Нажимаем на кнопку Try it Now.

Роутер может не определить тип нашего интернет соединения. Для того, чтобы использовать альтернативные типы подключения, требуется нажать внизу страницы кнопку Select Internet connection mode manually.

Откроется окно для выбора типа подключения. Выбираем Broadband acount online PPPoE и нажимаем Next.

Вводим данные по договору PPPoE: Broadband account – логин, Broadband Password – пароль. После чего нажимаем внизу кнопку Next.

Далее открывается настройка Wi-Fi, где можно задать SSID (имя сети) и Пароль. Если стоит галочка Make Wi-Fi password the router administrator password, то пароль от Wi-Fi установится в качестве пароля для доступа в веб-интерфейс роутера.

Настройка завершена.
