N - номер порта. Пример: (1/0/1), (0/0/1), (1/1).

V - номер VLAN.

Основная проверка порта.

show interface ethernet N - вывести статус N-го порта.

show mac-address-table interface ethernet N - вывести мак-адресса N-го порта.

virtual-cable-test interface ethernet N - статус кабеля N-го порта.

show running-config interface ethernet N - конфигурация N-го порта.

show logging buffered level warnings - просмотр журнала событий.

show interface ethernet status - вывести статус всех портов.

show interface brief - вывести статус всех портов.

show running-config - конфигурация всех портов.

Порядок изменения настроек порта.

Для изменения настроек порта сначала заходим в меню конфигурирования. Затем выбираем нужный порт. После внесения необходимых изменений, выходим из меню конфигурирования и сохраняем изменения. В некоторых моделях для входа в меню конфигурирования необходимо также прописать команду enable.

config - вход в меню конфигурирования.

interface ethernet N - выбор порта конфигурирования.

exit - выход в предыдущее меню конфигурирования.

write / Y - сохранение внесенных изменений.

Далее все команды выполняем в режиме конфигурирования!

Перезагрузка порта.

shutdown - отключение порта.

no shutdown - включение порта.

Основные команды настройки порта

switchport mode acces - перевод в режим абонентского порта.

switchport mode trunk - перевод в режим магистрального порта.

switchport access vlan V - присваивание VLAN на абонентском порту.

switchport trunk allowed vlan add V - добавление VLAN на магистральном порту.

switchport mac-address dynamic maximum (1-5) - настройка количества мак-адрессов.

loopback-detection specified-vlan V - включение функции распознавания петли.

loopback-detection control shutdown - функция отключения порта при распознавании петли.

storm-control broadcast 64 - ограничение штормового broadcast трафика до 64 Кбит/с.

storm-control multicast 64 - ограничение штормового multicast трафика до 64 Кбит/с.

speed-duplex auto - установка автоматического режима скорости и дуплекса.

speed-duplex force10-full - установка скорости 10 Мбит/с в режиме full дуплекс.

speed-duplex force100-full - установка скорости 100 Мбит/с в режиме full дуплекс.

speed-duplex force10-half - установка скорости 10 Мбит/с в режиме half дуплекс.

speed-duplex force100-half - установка скорости 100 Мбит/с в режиме half дуплекс.

*В случаях повреждения кабеля можно попробовать понизить скорость порта до 10 Мбит/с. Сначала пробуем режим full, затем half.
