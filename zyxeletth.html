N - номер порта.

Основная проверка порта.

show interface N - вывести статус N-го порта.

show mac address-table port N - вывести мак-адресса N-го порта.

cable-diagnostics N - статус кабеля N-го порта.

show running-config interface port-channel N - конфигурация N-го порта.

show logging - просмотр журнала событий.

show mac address-table all - вывести все мак-адресса.

show running-config - конфигурация всех портов.

Порядок изменения настроек порта.

Для изменения настроек порта сначала заходим в меню конфигурирования. Затем выбираем нужный порт. После внесения необходимых изменений, выходим из меню конфигурирования и сохраняем изменения. В некоторых моделях для входа в меню конфигурирования необходимо также прописать команду enable.

configure - вход в меню конфигурирования.

Далее выбираем режим конфигурирования.

interface port-channel N - конфигурирование порта.

vlan {vlan} - конфигурирование VLAN.

port-security - конфигурирование защиты порта.

exit - выход в предыдущее меню конфигурирования.

write / Y - сохранение внесенных изменений.

Команды ниже выполняем в режиме конфигурирования портаinterface port-channel N!

inactive - отключение порта.

no inactive - включение порта.

pvid {vlan} - присваивание VLAN.

speed-duplex auto - установка автоматического режима скорости и дуплекса.

speed-duplex 10-full - установка скорости 10 Мбит/с в режиме full дуплекс.

speed-duplex 100-full - установка скорости 100 Мбит/с в режиме full дуплекс.

speed-duplex 10-half - установка скорости 10 Мбит/с в режиме half дуплекс.

speed-duplex 100-half - установка скорости 100 Мбит/с в режиме half дуплекс.

*В случаях повреждения кабеля можно попробовать понизить скорость порта до 10 Мбит/с. Сначала пробуем режим full, затем half.


Команды ниже выполняем в режиме конфигурирования VLANvlan {vlan}!

fixed N - включает VLAN на порту.

forbidden N - включает VLAN на порту.


Команды ниже выполняем в режиме конфигурирования защиты портаport-security!

port-security N address-limit {1-5} - настройка количества мак-адрессов.
