N - номер порта. (От 0 до 3 или от 0 до 7)

ID - идентификатор порта. (от 0 до 127)

Сигнал до 22 - отличный.

Сигнал от 22 до 27 - хороший.

Сигнал выше 27 - плохой.

Основная проверка ONT.

show interface ont N/ID online - просмотр онлайн статуса.

show interface ont N/ID offline - просмотр оффлайн статуса.

show mac interface ont N/ID - вывести мак-адресса.

show interface ont N/ID connections - статистика переподключений.

show running-config interface ont N/ID - просмотр конфигурации.

show log ltp grep N/ID - просмотр журнала событий.

Дополнительная проверка ONT.

show interface ont N/ID ports - состояние подключения LAN-портов.

show interface ont N/ID state - состояние ONT.

show interface ont N/ID laser - параметры оптического модуля.

show interface ont N/ID rssi - уровень мощности сигнала.

Сброс и перезагрузка ONT.

send omci restore interface ont N/ID - сброс.

send omci reset interface ont N/ID - перезагрузка.

Обновление ПО.

show firmware ont - показать загруженные ПО.

schedule ont update N/ID {прошивка}ntu-rg-542x-2.5.8-build1068.fw.bin - запустить обновление ПО.

no schedule ont update N/ID - отменить обновление ПО.

show schedule ont update - просмотр списка задач обновления ПО.

clear schedule ont update - очистка списка задач обновления ПО.

Поиск ONT

show interface ont 0-7(3) online grep {серийный номер}ELTX85004455 - поиск по серийному номеру.

show interface ont 0-7(3) offline grep {серийный номер}ELTX85004455 - поиск по серийному номеру.

show mac interface gpon-port 0-7(3) include mac-address XX:XX:XX:XX:XX:XXCC:9D:A2:D5:CB:C4 - поиск по мак-адрессу.

Отключение и включение интерфейса(порта).

config - вход в меню конфигурирования

interface ont N/ID - выбор интерфейса конфигурирования.

shutdown - отключение порта.

do commit - сохранение изменений.

no shutdown - включение порта.

do commit - сохранение изменений.

Привязка ONT к интерфейсу.

config - вход в меню конфигурирования

interface ont N/ID - выбор интерфейса конфигурирования.

serial {серийный номер}ELTX85004455 - присваиваем серийный номер.

template {шаблон}ntu-rg(0-7)
BRIDGE(0-7) - присваиваем шаблон.

do commit - сохранение изменений.

Отвязка ONT от интерфейса.

config - вход в меню конфигурирования

no interface ont N/ID - очистка настроек интерфейса.

do commit - сохранение изменений.

Удаление пользователя.

acs - переходим в раздел конфигурирования ACS.

user - открываем подраздел USER.

delete user {логин}1910101
1105151 - удаляем пользователя.

commit - сохраняем изменения.

Удаление ONT.

acs - переходим в раздел конфигурирования ACS.

ont - открываем подраздел ONT.

delete ont {полный серийный номер}454C545885004455 - удаляем ONT.

commit - сохраняем изменения.

Первоначальная настройка

acs - переходим в раздел конфигурирования ACS.

ont - открываем раздел ONT.

show list include {полный серийный номер}454C545885004455 - находим ONT.(*копируем полный серийный номер).

exit - выходим до меню ACS.

user - открываем подраздел USER.

add user {логин} - создаем пользователя.

user {логин} - заходим в настройки пользователя.

set pon_serial {полный серийный номер}454C545885004455 - присваиваем серийный номер.

set profile {профиль}ntu-rg
ntu-2w
ntu1402
ntu5421
ntu5402
ntu5440 - присваиваем профиль.

set ppp_login {логин}1910101
1105151 - присваиваем логин.

set ppp_password {пароль}f3RoPguvrr - присваиваем пароль.

*Если дополнительно нужно настроить телефон, то дописываем следующие команды.

set voice1(2)_enable enabled - включаем телефонный порт.

set voice1(2)_number {номер}20110 - присваиваем номер.

set voice1(2)_password {пароль}f3RoPguvrrRjmgjYTKiR - присваиваем пароль.

commit - сохраняем изменения.

exit - выходим до меню ACS.

ont - открываем подраздел ONT.

ont {полный серийный номер}454C545885004455 - заходим в настройки ONT.

set profile {профиль}ntu-rg
ntu-2w
ntu1402
ntu5421
ntu5402
ntu5440 - присваиваем профиль.

commit - сохраняем изменения.

reconfigure - запускаем перенастройку ONT.

exit - выходим в главное меню.
